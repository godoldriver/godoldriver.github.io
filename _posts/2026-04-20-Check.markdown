## 6. 코드 블록 (Code Block)

### 인라인 코드

터미널에서 `jekyll serve` 명령어를 실행하면 로컬 서버가 시작됩니다.

### 블록 코드 — Bash

```bash
# Jekyll 로컬 서버 실행
bundle exec jekyll serve

# 빌드만 할 경우
bundle exec jekyll build
```

### 블록 코드 — YAML (_config.yml 예시)

```yaml
title: "나의 블로그"
description: "GitHub Pages + Jekyll 블로그"
author:
  name: "홍길동"
  email: "hello@example.com"

# 테마 설정
remote_theme: mmistakes/minimal-mistakes

# 플러그인
plugins:
  - jekyll-feed
  - jekyll-sitemap
```

### 블록 코드 — Markdown 포스트 헤더

```markdown
---
layout: single
title: "포스팅 제목"
date: 2026-04-20 09:00:00 +0900
categories: [카테고리]
tags: [태그1, 태그2]
---

본문 내용을 여기에 작성합니다.
```

### 블록 코드 — Python

```python
def greet(name: str) -> str:
    """이름을 받아 인사말을 반환합니다."""
    return f"안녕하세요, {name}님! 블로그에 오신 것을 환영합니다."

if __name__ == "__main__":
    print(greet("대장님"))
```
